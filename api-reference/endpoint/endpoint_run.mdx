### Description

Any pipeline can be deployed to an [Endpoint](/endpoints/introduction), so that it can be accessed externally through an API.
This endpoint is used to call a deployed Endpoint.

You can send request with various inputs. Inputs can be of any type such as string, array of strings, or ChatMessage [(Check OpenAPI schemas)](https://github.com/skull8888888/docs/blob/main/api-reference/openapi.json).

The response is an object. Its keys are the names of the output nodes, and the values are of type Message [(Check OpenAPI schemas)](https://github.com/skull8888888/docs/blob/main/api-reference/openapi.json).
