## What is it

A Laminar project is a collection of [pipelines](/pipeline/introduction), [datasets](/datasets/introduction),
[evaluations](/evaluations/introduction), and [traces](/tracing/introduction).
Access [API keys](/api-reference/introduction#authentication) and environment variables are configured at the project level as well. Think about it as a scope of a certain work. A project always belong to a workspace.

## Creating a project

1. [Optional] create a new workspace.
1. Navigate to the projects page by clicking the Laminar's logo in the top left corner, once in any (non-landing) page.
1. Click "Add project".
1. Select the workspace the project will belong to.
1. Give the project a name.
1. Click "Create".
