Laminar events are fully-compatible with OpenTelemetry events. They are stored in the `events` field of the span object.

## Event object

| Attribute | Description | Type | Laminar representation (if different) | Example |
| --- | --- | --- | --- | --- |
| name | Name of the event | string | | `my_event` |
