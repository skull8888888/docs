## Laminar

Ship reliable LLM agents and applications.

We provide a single platform to build, deploy, monitor, and evaluate complex LLM agents.

## Key features

- **Versatile [pipeline builder](/documentation/pipeline/introduction)**: Build LLM pipeline using variety of nodes with required functionality provided out of the box.
- **[Semantic Search](/documentation/nodes/semantic-search)**: Cache data or upload it, so that it can be quickly accessed based on semantic similarity. Fully hosted by us.
- **[Evaluations](/documentation/evaluations/introduction)**: Evaluate your LLM pipeline with custom evaluator pipelines.
- **[Datasets](/documentation/datasets/introduction)**: Store your data for use in evaluations or for semantic retrieval.
- **[Endpoints](/documentation/endpoints/introduction)**: Deploy your pipeline to an API endpoint in seconds to our scalable inherently fast backend.
- **[Logging](/documentation/logs/introduction)**: Conveniently access logs and see how users interact with your pipeline through endpoints.

To see all features, you can read more in other sections.
